# Simulateur de Craps Complet

Jouez au Craps avec toutes les mises classiques !

??? abstract "Règles du Craps"

    Le Craps est un jeu de dés avec deux phases principales :

    ### Phase Come Out Roll
    - **7 ou 11** : Pass Line gagne, Don't Pass perd
    - **2, 3 ou 12 (Craps)** : Pass Line perd, Don't Pass gagne (sauf 12 = push)
    - **Autre numéro** : Devient le "Point"

    ### Phase de Point
    - **Point refait** : Pass Line gagne
    - **7 (Seven-Out)** : Don't Pass gagne, fin du tour
