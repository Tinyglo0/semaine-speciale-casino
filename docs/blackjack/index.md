# Jeu de Blackjack

!!!tip "Règles du jeu"
    - L'objectif est d'obtenir 21 points ou de se rapprocher le plus possible sans dépasser
    - Les figures (J, Q, K) valent 10 points
    - L'As vaut 11 ou 1 point selon votre avantage
    - Le croupier tire jusqu'à 17 points minimum
